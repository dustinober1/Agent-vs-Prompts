# Chapters

This folder contains per-chapter working drafts/stubs derived from `BOOK_OUTLINE.md`.

## How to use
- Start from `chapters/_chapter_TEMPLATE.md` for new chapters.
- Each chapter stub should link back to the outline and include:
  - purpose + reader takeaway
  - key points
  - case study thread (Research+Write and Instructional Design)
  - artifacts and exercises

## Case study anchors
- Research+Write: `case_studies/research_write_policy_change_brief_TEMPLATE.md`
- Instructional Design: `case_studies/instructional_design_compliance_training_MODULE_TEMPLATE.md`
